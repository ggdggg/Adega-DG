Sobre
Minha Adega é um PWA — um app web que funciona offline.

Para instalar no iPhone:
1. Abra esta página no Safari
2. Toque no botão Compartilhar (quadrado com seta)
3. Role e escolha "Adicionar à Tela de Início"
4. Pronto — vira um ícone no seu celular

Os dados ficam salvos no dispositivo. Use "Exportar" para fazer backup.
